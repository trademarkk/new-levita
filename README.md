# LEVITA CLUB — клиентская страница

Адаптивный лендинг по материалам памятки LEVITA CLUB.

## Локальный запуск

```bash
npm install
npm run dev
```

## Публикация на Vercel

1. Загрузите проект в GitHub/GitLab или выберите папку через Vercel CLI.
2. Импортируйте проект в Vercel.
3. Настройки определятся автоматически: команда сборки `npm run build`, папка результата `dist`.

Переменные окружения не требуются.
